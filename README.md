## 環境配置
1. 下載Ollama
2. 下載程式碼
3. 安裝模組，請先安裝Pytorch和其核心的依賴  
`pip install torch torchvision torchaudio`
4. 接下來安裝其餘模組  
`pip install -r requirements.txt`
5. 安裝ffmpeg
6. 僅須安裝qwen 4b如果沒有，程式應該會幫你裝
## 開始使用
1. 執行run_main.bat這個檔案
2. 進入GUI介面選擇聊天用模型和語音轉文字模型
| 建議使用的聊天模型      | 建議使用的文字轉語音模型 |
| ----------------------- | ------------------------ |
| qwen3b:30b              | large-v3(多語言)         |
| qwen3b:8b               | medium(多語言)           |
| qwen3b:4b               | medium.en(英文)          |
